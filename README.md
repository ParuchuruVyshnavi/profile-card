# 🌟 Modern Developer Profile Card

A modern, responsive, and animated developer profile card built using **HTML, CSS, and JavaScript**. The project features a premium glassmorphism UI, animated neon effects, dark/light theme switching, and interactive animations.

---

## ✨ Features

- 🌙 Dark & Light Theme Toggle
- 💾 Theme Preference Saved using LocalStorage
- 💎 Glassmorphism UI Design
- 🔄 Animated Rotating Neon Profile Ring
- 🌈 Animated Gradient Name
- 💻 Animated Developer Badge
- 📱 Fully Responsive Design
- 🚀 Gradient Follow Button
- ✉️ Message Button
- 🌐 Social Media Links
- 💡 Animated Skill Cards
- 📊 Progress Bars
- ✨ Ripple Button Effect
- 🎯 Hover Animations
- 🖱️ 3D Card Tilt Effect
- ⚡ Smooth Transitions

---

## 🛠️ Built With
